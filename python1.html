URLconf 설정하며 어려웠던 부분.

*** url 관리를 2개의 urls.py로 나눠서 관리한다. ***

처음 URLconf를 코딩할때
urlpatterns = patterns('',
     url(r'^admin/', include(admin.site.urls)),
)


이런식으로 되어있었는데, url오류가 자꾸 발생함. 왜그런지 찾아보니 patterns는 예전버전에서 사용. (내장함수호출로 python 3 에서도 사용가능)
python 3 에서는patterns 없이 사용 가능.

mysite   -->   urls.py
from django.conf.urls import include, url    -->    django 내장함수인 include와 url을 호출

from django.contrib import admin    -->    django모듈에서 제공되는 인증 프레임워크 (admin페이지 접속시 편집가능한 그룹과 사용자같은 몇종류의 컨텐츠를 볼수 있음.)


url(r'^admin/', admin.site.urls)    -->    admin url을 지정해줌.

url(r'', include ('polls.urls', namespace="polls")),    -->    polls\urls.py 파일을 포함한다는 의미.

polls   -->   urls.py
from django.conf.urls import include, url    -->    django 내장함수인 include와 url을 호출

form polls import views    -->    polls폴더 안에있는 views.py 를 호출


각각의 url을 호출.
