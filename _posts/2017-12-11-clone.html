git clone 해보기!


집에서 git, python, ruby, jekyll, bundle 등등을 설치하여 원래 사용하던 저장소 데이터를 복사하기 위해 찾던중 clone 발견 시도해 보았다.

처음 git clone https://github.com/lsk325/lsk325.github.io 로 시도 했으나 오류발생.. 회사에서는 잘됐었는데 집에서는 오류를 뱉어 냈다.

구글링중 git clone git://github.com/lsk325/lsk325.github.io 로 clone을 당겨오는게 있어서 시도해 봤다.

성 공 !
