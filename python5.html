♣ 클래스? (class)

• 변수나 함수들을 모아서 필요할 때 사용할 수 있는 집합체
• 간단히 설명하면 변수와 함수를 모아 넣은 것!
• 클래스명은 PEP 8 Coding Convention에 가이드된 대로 각 단어의 첫 문자를 대문자로 하는   CapWords 방식으로 명명한다.

♣ 메소드? (method)

• 함수와 비슷한 개념
• 클래스에 묶여서 클래스 안에 있는 인스턴스와 관계되는 함수

♣ 인스턴스? (instance)

• 클래스에 의해 만들어진 객체
• 인스턴스는 각자 자신의 값을 가지고 있음

♣ 프로퍼티? (Property)

• 변수(Property)

♣ 모델링? (modeling)

• 클래스로 현실의 개념을 표현

♣ 문제점? (problem)

class를 사용하여 객체를 쓰게되면 유용해보이지만.
하나의 class로 선언된 객체의 변수를 밖에서 마음껏 바꿀수 있다는 큰 문제가 있다!

이를 없애기 위해서 파이썬에서 class 내부의 변수를 나타낼 때에는 __를 먼저 입력한 후 변수를 선언하게 된다.
그렇게 되면 이에 대한 값은 내부에서만 변경시킬 수 있고 외부에서는 접근을 할 수 없게 된다.

만약, 사용자가 임의로 바꾸고 싶을 때는?
위의 방법을 사용하지 않고 외부에서 마음껏 접근을 하게 되면 큰 문제가 발생하므로 함수를 통해서 변수를 바꿀 수 있도록 접근을 돌려서 할 필요가 있다.
이러한 방법에 사용되는 함수가 get과 set 함수이다.
